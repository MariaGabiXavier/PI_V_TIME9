# Sched

**Sched** é uma plataforma inteligente voltada a minimercados, mini padarias e pequenos comércios alimentares. O foco do sistema é o apoio à tomada de decisão, proteção do fluxo de caixa e a redução drástica do desperdício de alimentos através de previsões de demanda baseadas em IA.

O projeto foi desenvolvido como parte do componente curricular **Projeto Integrador V - Engenharia de Software** da **PUC-Campinas**.

## 💡 Introdução

Pequenos negócios alimentares são essenciais para a economia local, mas frequentemente enfrentam dificuldades na gestão eficiente de estoque. O **Sched** utiliza **Inteligência Artificial** para transformar dados históricos de vendas em previsões de demanda assertivas, permitindo que o pequeno comerciante tome decisões baseadas em dados, não apenas em intuição.

> * **Por que:** Apoio tecnológico acessível para reduzir desperdícios e proteger o caixa de pequenos empreendedores.
> * **Como:** Plataforma com IA para análise de risco de vencimento e predição de demanda.
> * **O que:** Um sistema web inteligente com gestão de estoque, controle de validade e indicadores.

---

## ⚠️ Problema

* **Gestão:** Decisões de reposição tomadas sem dados históricos estruturados.
* **Impacto Financeiro:** Excesso de estoque imobiliza capital, enquanto a ruptura causa perda de vendas.
* **Desperdício:** O vencimento de produtos perecíveis gera prejuízo financeiro, ambiental e social.

---

## 🎯 Objetivo

Desenvolver uma plataforma inteligente para apoio à gestão de estoque, utilizando Machine Learning para predição de demanda e análise de risco de desperdício.

### Objetivos específicos:
* Projetar arquitetura **SaaS multi-tenant** (isolamento de dados por empresa).
* Implementar modelos de **Machine Learning** para regressão de demanda.
* Avaliar o desempenho dos modelos com métricas **MAE** e **RMSE**.
* Criar um **dashboard interativo** com indicadores de estoque e risco de desperdício.

---

## ⚙️ Funcionalidades Principais

✅ **Predição de Demanda** – Previsões para períodos de 7, 15 ou 30 dias usando IA.
✅ **Gestão de Validade** – Registro de data de validade associada ao lote para produtos perecíveis.
✅ **Alertas Inteligentes** – Identificação de baixo estoque e risco de vencimento.
✅ **Importação de Dados** – Suporte para carga de grandes volumes via CSV ou XML
✅ **Recomendações** – Sugestões automáticas para reposição de estoque.
